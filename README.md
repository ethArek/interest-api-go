# Interest API in Go using docker(-compose)

## Requirements:

- Docker and docker-compose

## Building and starting app

`$ docker-compose up --build -d`

## Starting app

`$ docker-compose up -d`

## Deleting running container

`$ docker ps`
`$ docker stop <container_id>`
`$ docker rm <container_id>`
